Ordinary-Least-squars - einfache Lineare Regression

In der linearen Regression haben wir eine unabhängige Variable, die sich auf eine abhängige Variable beziehen kann mit der Formel:

lineare Formel: $t=y-Achsenschnittpunkt$ $m=Steigung der Geraden$ $y=m x + t$

Lineare Regression:

$b_{0}=y-Achsenschnittpunkt$ $b_{1}+b_{1}=SteigungderGeraden$ $\hat{y}=b_{0}+b_{1} x$

Ziel der Regression ist es, die Gerade zu finden, die unsere Daten am besten beschreibt = den kleinsten Fehler hat.

Unser Ziel ist es, den Wert einer abhängigen Variablen (y) auf der Basis einer unabhängigen Variablen (x) vorherzusagen.

Eigene Erklärung -OLS

es wird versucht eine Gerade durch die Datenwolke zu legen, welche den Fehler minimiert. Der fehler ist zunächst die Abweichung eines Punktes von der Geraden. Wird dieser Fehler quadriert, so heben sich negative und positive Fehlerabweichungen gegenseitig auf. Dabei entsteht auch die Bezeichnung Quadrat(A*A). vgl MSE / RMSE .

BSP - lineare Regression

Ein Manager möchte die Beziehung zwischen der Anzahl der Stunden, die eine Anlage pro Woche in Betrieb ist, und der wöchentlichen Produktion herausfinden.

  • Hier ist die unabhängige Variable x die Betriebsstunden und die abhängige Variable x ist das Produktionsvolumen.

Lösung:

  1. Tabelle und Grafik anlegen

  2. Erstellen die Kalkulation. vgl. Händische berechnung

    $b_{1}=\frac{\sum\left(x_{i}-\bar{x}\right)\left(y_{i}-\bar{y}\right)}{\sum\left(x_{i}-\bar{x}\right)^{2}}=\frac{558}{124}=\mathbf{4} . \mathbf{5}$ $b_{0}=\bar{y}-b_{1} \bar{x}=134-(4.5 \times 40)=-\mathbf{4 6}$ $\hat{y}=-\mathbf{4} 6+\mathbf{4} . \mathbf{5} \boldsymbol{x}$

  3. Basierend auf dieser Formel ( $\hat{y}=-\mathbf{4} 6+\mathbf{4} . \mathbf{5} \boldsymbol{x}$ ), kann man nun berechnen, wie lange die Anlage laufen sollten, wenn der Manager 125 Einheiten pro Woche produzieren möchte:

    $\hat{y}=b_{0}+b_{1} x$ $125=-46+4.5 x$ $x=\frac{171}{4.5}=38$ Stunden pro Woche

Python-Implementierung

from scipy.stats import linregress
x = [34, 35, 39, 42, 43, 47]
y = [102, 109, 137, 148, 150, 158]
slope = round(linregress(x,y).slope,1)
intercept = round(linregress(x,y).intercept,1)
print(f'y = {intercept} + {slope}x')
y = -46.0 + 4.5x
