Die SI_SECTIONINDEX_TEXT im Content Directory können zuvor mit den klassischen Mitteln der Markdown Technologien genutzt und bearbeitet werden. Diese Änderungen werden dann in den SectionIndex.html Dateien übernommen.

  • vorgelagerte SectionIndex (= Singular, es kann nur eine vorgelagerte Section geben)

    • Name
    • Beschreibung

    • Name
    • Beschreibung
