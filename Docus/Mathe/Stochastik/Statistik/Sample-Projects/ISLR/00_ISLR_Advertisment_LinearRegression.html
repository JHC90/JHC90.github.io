Basics Lineare Regression¶

das hier ist aus Kapitel 2 des ISLR

Basic-Understanding der Regression OLS:

  • Artikel 1

Ziel der Linearen Regression ist einen Zusammenhang zwischen zwei Variablen aufzudecken. Wir starten mit zwei Variabeln(einfache lineare Regression // nicht mutliplelineare Regression = >=2 Input Feature)

Als Datensatz wird hier der Advertisment von ISLR verwendet. Der datensatz zeigt die Ausgaben von TV, Radio und newspaper und wie diese sich auf sales auswirken. in diesem Notebook betrachte ich nur ein feature(tv), alle Feature zusammen werden in dem Notebook Multiple Regression betrachtet.

p-Value¶

Der P-Value gibt aufschluss wie sehr die Daten stark miteinander korrelieren. Der P-Wert soll so niedrig wie möglich sein. Je niedriger ein p-Wert desto eher erklärt dieses Feature das Target. Daumenregel: P-Wert eines feature soll kleiner als 0.05 sein

R² -Value¶

Der R²(R-squared) ist bei 0.612, das bedeutet das ~61% der Variablität der von Sales wird somit erklärt. Somit kann davon ausgegangen werden, somit werden die anderen Variablen (newspaper & radio) wohl auch eine Auswirkung haben.
