Data Loading & Exploration¶

Natürlich beginnen wir unseren Machine Learning Prozess mit dem Laden der Daten. Hier verwenden wir pandas. Zum einen bietet Pandas eine Menge an Methoden, die uns beim Laden und Selektieren der Daten helfen; zum anderen aber bietet uns pandas auch einigen Support, bei der Vorverarbeitung der Daten.

In [35]:
dataset_location ="./titanic.csv"
dataset = pd.read_csv(dataset_location)

Das titanic Datenset enthält bereis ein Zielattribut (survived), das wir zum Trainieren eines Klassifizierers verwenden können. Unterhalb legen wir die input- und output-Features fest und teilen unserer Datenset gleich in Trainings- und Testdaten. Die Testdaten legen wir gleich beiseite und rühren sie bis zur Modellevaluation nicht mehr an.

In [6]:
output_features = [
    "survived"
]

input_features = [
    "pclass",
    "name",
    "sex",
    "age", 
    "sibsp",
    "parch",
    "ticket",
    "fare",
    "cabin",
    "embarked",
    "boat",

]

train_X, test_X, train_y, test_y = train_test_split(
    dataset[input_features], 
    dataset[output_features],
    test_size=0.2
)

Unterhalb ist die finale Form der Input/Output Matrix des Trainings- und Testdatensets abgebildet. Da wir mit Pipelines weiterarbeiten, folgender Hinweis: Längenwirksame Operationen innerhalb der Pipeline (z.B. dropna()) müssen auch am Output-Vektor durchgeführt werden, weil sonst Input und Output unterschiedliche lang sind. Solltet ihr Vorhaben solche Operationen vorzunehmen, ist es klüger, das Output-Feature entlang der Pipeline mitzuführen, denselben Operation zu unterziehen und die Input/Output-Trennung am Ende der Pipeline vorzunehmen.

import numpy as np

def pretty_confusion(target, prediction):
    """Prettify the on-board confusion matrix of sklearn."""

    cmc = ["Condition positive", "Condition negative"]
    cmi = ["Predicted condition positive", "Predicted condition negative"]

    matrix = confusion_matrix(target, prediction)
    cm = matrix.astype('float') / matrix.sum(axis=1)[:, np.newaxis]
    return pd.DataFrame(cm , columns=cmc, index=cmi)


def debug_print(X, debug):
    """This method prints some debug information based on a debug parameter."""
    if not debug in ["input", "shape", "columns", False]:

        print(X.head/(5))
    elif debug=="shape":

    elif debug is False:

Feature Engineering mit Pipeline-Transformatoren¶

Wir werden die Vorverarbeitunsschritte in diesem Projekt mit Hilfe von sklearn Pipelines lösen. In diesen Pipelines können wir sämtliche sklearn Transformatoren und Estimatoren verwenden, die eine fit und ´transform´/´predict´ methode besitzten. Mit diesen Bausteinen können wir bereits viel erreichen. Um zusätzliche Funktionen in eine Pipeline einzubauen, können wir aber außerdem eigene Transformatoren/Estimatoren nach folgendem Vorbils entwickeln.

class FeatureSelector:
    """This transformer lets you pick columns from a pandas dataset based on name"""
    def __init__(self, features = [], debug=False):
        self.d = debug
        self.colnames = None
        if type(features) != list:
            raise ValueError("Input features must be of type List.")
        elif type(features) == list:
            self.columns = features

    def fit(self, X, y=None):
        return self

    def transform(self, X):
        debug_print(X=X, debug=self.d)
        X = X[self.columns]
        self.colnames = X.columns.tolist()
        return X

    def get_feature_names(self):
        return self.colnames

class EmbarkedImputer():
    """Imputes values into Embarked variable."""
    def __init__(self, debug=False):
        self.d = debug
        self.colnames = None

    def fit(self, X, y=None):
        return self 

    def transform(self, X):
        debug_print(X=X, debug=self.d)
        df = X.copy()        
        value_to_input = df.loc[(df['fare'] < 85) & (df['fare'] > 75)  & (df['pclass'] == 1)]['embarked'].mode()
        value_to_input = value_to_input[0]
        df["embarked"] = df["embarked"].fillna(value_to_input)
        self.colnames = df.columns.tolist()
        return df

    def get_feature_names(self):
        return self.colnames

class NameToTitle:
    """Replaces the name with the person's title"""
    def __init__(self, debug=False):
        self.d = debug
        self.colnames = None

    def fit(self, X, y=None):
        return self 

    def transform(self, X):
        debug_print(X=X, debug=self.d)
        df = X.copy()
        title_dictionary = {
            "capt":"Officer", 
            "col":"Officer", 
            "major":"Officer", 
            "dr":"Officer",
            "jonkheer":"Royalty",
            "rev":"Officer",
            "countess":"Royalty",
            "dona":"Royalty",

            "mme":"Mrs",
            "ms":"Mrs",
            "mrs":"Mrs",

from sklearn.preprocessing import OneHotEncoder

class CustomOneHotEncoder:
    """OneHotEncoder that passes a labeled pandas dataframe"""
    def __init__(self, debug=False):
        self._encoder = None
        self._column_list=[]
        self.d = debug
        self.colnames = None

    def fit(self, X, y=None):
        self._encoder = OneHotEncoder(sparse=False)
        self._encoder.fit(X)
        return self

    def transform(self, X):
        self._column_list=[]
        debug_print(X=X, debug=self.d)
        result_X= self._encoder.transform(X)
        for column in self._encoder.categories_:
            for value in column:
                self._column_list.append(value)
        X = pd.DataFrame(result_X, columns=self._column_list)
        self.colnames = X.columns.tolist()
        return X

    def get_feature_names(self):
        return self.colnames

        self.colnames = None

    def fit(self, X, y=None):
        if "sibsp" not in X.columns:
            raise ValueError("X does not contain sibsp")
        if "parch" not in X.columns:
            raise ValueError("X does not contain parch")
        return self

    def transform(self, X):
        X["travel_alone"] = None
        X["travel_alone"] = X.loc[
            (X["sibsp"] == 0) & (X["parch"] == 0)
        ]["travel_alone"].map(lambda x: 1)
        X["travel_alone"] = X["travel_alone"].fillna(0)
        self.colnames = X.columns.tolist()
        return X

    def get_feature_names(self):
        return self.colnames

Data Pre-Processing mit Pipeline & FeatureUnion¶

Pipelines können verwendet werden, um einzelne Bearbeitungsschritte sequenziell aneinander zu reihen. Das ist die zentrale Funktion der Pipeline Klasse. Die fit Methode der Klasse ruft nacheinander alle fit Methoden der einzelnen Transformatoren/Estimatoren auf. Die predict Methode ruft die transform Methode aller Transformatoren und die predict Methode aller Estimatoren auf.

Die FeatureUnion Klasse erlaubt es außerdem, in eine Pipeline mehrere, parallele Pipelines einzubauen. Die Ergebnisse aller Pipelines müssen gleich viele Zeilen liefern und werden am Ende der FeatureUnion Spalte für Spalte aneinander gefügt. Unterhalb haben wir die aktuelle Pipeline schematisch dargestellt.

In [24]:

        ("kid_with_two_parents_mapper", KidWithTwoParentsMapper(age=18)),
        ("travel_alone_adder", TravelAloneMapper()),
        ("age_imputation", AgeImputer(debug=False)),
        ("age_binner", AgeBinner(debug=False)),
        ("impute_embarked", EmbarkedImputer(debug=False)),
        ("embark_mapper", EmbarkMapper(debug=False)),
        ("calc_passenger_fare", PassengerFareCalculator(debug=False)),
        ("name_to_title", NameToTitle(debug=False)),
        ("feature_union",
            FeatureUnion(
                [
                    ("num_pipe", FeaturePipeline(
                        [
                            ("numeric_selector", FeatureSelector(["parch","sibsp"], debug=False)), #"fare",
                            ("median_imputation", CustomSimpleImputer(strategy="median")),
                            ("robust_scaler", CustomRobustScaler()),
                        ]
                    )),
                    ("non_num_pipe", FeaturePipeline(
                        [
                            ("non-numeric_selector", FeatureSelector(["age", "embarked", "sex", "name"], debug=False)),
                            ("one_hot_encoder", CustomOneHotEncoder()),
                        ]
                    )),
                    ("bypass_pipe", FeaturePipeline(
                        [
                            ("bypass_selector", FeatureSelector(["pclass", "kid_with_two_parents", "travel_alone"], debug=False)),
                        ]
                    ))
                ]
            )
        )
    ]
)
